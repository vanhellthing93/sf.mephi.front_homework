## Установка

1. Клонируйте репозиторий:
    ```bash
    git clone https://github.com/vanhellthing93/sf.mephi.front_homework
    ```
2. Перейдите в директорию проекта:
    ```bash
    cd sf.mephi.front_homework
    ```
3. Откройте `index.html` в вашем браузере.
